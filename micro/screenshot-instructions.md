# Screenshot Placeholder

Add a `screenshot.png` file here for your child theme.

## Specifications

- **Dimensions:** 1200 x 900 pixels (4:3 ratio)
- **Format:** PNG or JPG
- **Content:** Preview of your theme design
- **Shows:** In WordPress Admin → Appearance → Themes

## Quick Create

If you don't have a custom screenshot, you can:

1. **Take a screenshot** of your site's homepage
2. **Resize** to 1200x900 pixels
3. **Save** as `screenshot.png` in the child theme root
4. **Refresh** WordPress theme page

The screenshot helps you visually identify the theme in the WordPress admin panel.
